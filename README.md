📅 Slot Booking Website
A modern and responsive slot booking web application built with HTML, CSS, and JavaScript. This project allows users to easily select and reserve available time slots for appointments or services.

🚀 Live Demo
🔗 https://mikadoai0.github.io/Slot-wise/

🧩 Features


⏰ Time Slot Selection
Users can view and select available time slots.


🚫 Prevent Double Booking
Once a slot is booked, it becomes unavailable.


📝 Booking Form
Users provide their name and email before confirming a booking.


💾 Local Storage Integration
Booked slots are saved in the browser and persist after page reload.


✅ Booking Confirmation
Instant feedback after a successful booking.


🔄 Reset Functionality
Option to clear all bookings and start fresh.


📱 Responsive Design
Fully optimized for mobile, tablet, and desktop devices.


🎨 Modern UI/UX
Clean layout, smooth animations, and SaaS-inspired design.



🛠️ Tech Stack

HTML5 – Structure
CSS3 – Styling & responsiveness
JavaScript (Vanilla JS) – Functionality & logic

⚙️ How It Works

User selects an available time slot
Fills in name and email
Submits the booking
Slot becomes unavailable
Data is stored in localStorage
